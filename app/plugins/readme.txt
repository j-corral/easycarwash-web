Intro :
Un plugin sera un module complémentaire facultatif qui permettra de modeler le Framework
en outil clef en main adapté aux besoins de chaque projet.
Ainsi, avec le framework de base et le bon plugin, on peut s'adapter au projet sans changer de technologie.


-----------

- Blog
